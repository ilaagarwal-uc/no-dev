## Usage Guidelines

**Primary Accent**: `#97B3AE` (Muted Teal)
- Buttons
- Links
- Interactive elements
- Focus states
- Titles

**Secondary Accents**: `#F2C3B9` (Soft Rose)
- Hover states
- Highlights
- Loading indicators

**Backgrounds**:
- Main: `#F0EEEA` (Off-white)
- Light: `#F0DDD6` (Warm Beige)
- Subtle: `#D2E0D3` (Soft Sage)

**Text**:
- Primary: `#2d3748` (Soft Charcoal) - Keep existing
- Secondary: `#78716c` (Warm Gray) - Keep existing

**Borders & Dividers**: `#D6CBBF` (Taupe)

## Implementation Rules

1. Use #97B3AE for all primary interactive elements
2. Use #F2C3B9 for hover and loading states
3. Use #F0EEEA as main background
4. Use #D2E0D3 for subtle background variations
5. Use #D6CBBF for borders and dividers
6. Keep text colors unchanged (#2d3748, #78716c)
7. No gradients - solid colors only
8. Maintain smooth animations and transitions
