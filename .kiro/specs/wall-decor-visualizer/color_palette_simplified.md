# Simplified Color Palette

## Core Colors (Minimal Approach)

**Primary Accent**: `#d4a5a5` (Muted Dusty Rose)
- Buttons
- Links
- Interactive elements
- Hover states
- Focus states

**Background**: `#fafaf9` (Warm Off-white)
- Main page background
- Default surface

**Surface**: `#f5ede7` (Ultra-light Warm)
- Cards
- Form containers
- Sections

**Text**: `#2d3748` (Soft Charcoal)
- All primary text
- Headings

**Text Secondary**: `#78716c` (Warm Gray)
- Secondary text
- Descriptions

**Border**: `#e7e5e4` (Warm Light Gray)
- Dividers
- Input borders
- Card borders

**Loading**: `#C1E1C1` (Soft Mint Green)
- Loading spinners only
- Progress bars only

## Usage Rules

1. **One primary color** (#d4a5a5) for all interactive elements
2. **No gradients** - use solid colors only
3. **No secondary accent colors** - keep it simple
4. **Loading states only use mint green** (#C1E1C1)
5. **Warm off-white background** throughout
6. **Minimal color variation** - focus on typography and spacing

## Implementation

- Remove all multi-color gradients
- Use solid #d4a5a5 for buttons, links, focus states
- Use #C1E1C1 only for loading indicators
- Keep backgrounds consistent (#fafaf9 and #f5ede7)
- Rely on animations and spacing for visual hierarchy
